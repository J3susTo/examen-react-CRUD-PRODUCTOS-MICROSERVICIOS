## Funcionalidades implementadas (100 % cumplidas)
- Login con JWT
- Listado de productos con búsqueda
- Crear producto
- Editar producto (modal reutilizado)
- Eliminar producto con confirmación
- Token enviado automáticamente en headers
- Protección de rutas (sin login no entra)
- Interfaz moderna y responsive con Bootstrap 5
- Manejo de errores y loading states

## Microservicios necesarios (deben estar corriendo localmente)
- ms-auth → http://localhost:8081/apis/codigo/auth
- ms-productos → http://localhost:8082/apis/codigo/productos

## Tecnologías usadas
- React 18 + Vite
- React Router v6
- Axios
- Bootstrap 5 + React Bootstrap
- Vercel (deploy frontend)
- Spring Boot + JWT (backend)
